# ATM Simulation System

A console-based ATM simulation written in x86 Assembly, built as a Computer Organization and Assembly Language (COAL) course project.

## About

This project simulates basic ATM functionality at the assembly level, handling user interaction, PIN verification, balance checking, deposits, and withdrawals — all implemented using low-level x86 instructions.

## Features

- PIN-based login
- Check account balance
- Deposit funds
- Withdraw funds
- Simple console UI

## Requirements

- [MASM](https://www.masm32.com/) or any compatible x86 assembler
- Visual Studio (solution file included) with MASM integration

## Running the Project

1. Clone the repo:
   ```bash
   git clone https://github.com/HuzaifaAhmedBari/ATM-Simulation-System-Coal-Project.git
   ```
2. Open `ATM Simulation System.sln` in Visual Studio.
3. Build and run.
